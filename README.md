Local Running instructions:
Prerequisites:
- Node JS
- NPM

1. Clone repo, using the URL to the repo

git clone https://github.com/KWilson-Dev/wad2-cw-kw.git



cd wad2-cw-kw

2. Install dependences:
   npm install
3. Seed databases
   npm run seed
4. Start application:
   npm start
   Or
   Node index
